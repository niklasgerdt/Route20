Event-Based Routing System on JVM. Provides PubSub-model. Supports blocking and Non-blocking routing.

###Rationale
- loose coupling
- scalability
- adaptivity

####Building blocks
- Event (Header, Type, Payload)
- Producer
- Consumer
- Notification service
